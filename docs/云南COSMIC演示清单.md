# 云南 COSMIC 演示清单

## 演示定位

`YN云南COSMIC` 是根据《中国移动云南公司2024年资产安全管理平台研发项目-评估结果》生成的前端 Mock 演示模块，用于展示功能点覆盖情况。

本模块不连接真实后端，页面数据、流程状态、导入导出、审批、接口调用、稽核校验和统计结果均为前端模拟效果。

## 启动方式

```bash
npm install
npm run serve
```

默认访问地址：

```text
http://localhost:8888
```

## 菜单路径

```text
资产信息上报调整 / YN云南COSMIC
```

当前页面按 `功能用户需求` 展示。每个页面内的按钮对应 Excel 中的触发事件和功能过程。

## 通用按钮演示口径

| 按钮类型 | 演示说明 |
| --- | --- |
| 新增/录入/配置 | 打开表单弹窗，确认后新增模拟数据 |
| 修改/编辑/变更 | 打开编辑弹窗，确认后更新模拟数据 |
| 删除 | 弹出确认框，确认后模拟删除 |
| 查询/查看/展示 | 刷新表格或打开详情弹窗 |
| 导入 | 打开导入弹窗，生成模拟导入记录 |
| 导出 | 模拟导出 Excel 或提示导出成功 |
| 审批/审核/驳回 | 展示模拟审批节点和处理意见 |
| 下发/发送/上报/同步 | 展示模拟接口调用日志 |
| 接收/反馈/返回 | 展示模拟回执和结果 |
| 稽核/校验/匹配/对比 | 展示模拟通过项和异常项 |
| 分析/统计/监控 | 展示模拟统计结果或图表 |

## 验收关注点

验收时建议重点确认：

- 菜单可进入 `YN云南COSMIC`。
- 重点页面可打开，无空白页。
- 表格有数据。
- 查询、重置、刷新有反馈。
- 功能过程按钮可点击。
- 弹窗能展示业务含义、流程步骤或处理结果。
- 导入导出有演示反馈。
- 审批、下发、稽核、统计等流程能闭环展示。
